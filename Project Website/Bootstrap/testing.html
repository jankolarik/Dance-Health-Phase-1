Testing Method

For each new method we create, we test its functionality to make sure it produce a reliable result.

Kinect - Calibration

Calibration method is used to calculate the relative height of the person's feet compared to camera position, so that we can use it as floor height and calculate joints data relative to floor instead of camera position.

Testing method: having a person stand in the same floor whit the camera position fixed, make this person stand in differnt places and comapred the calibration result.

Test result: So far, this is accurate, we will continue testing this feature as we develop the project.

Kinect - Activity Level

Activity level is a method that calculate the average displacement of each joint moved between per frame, we can use this result to determine how active the person in front the camera is.

Test method: having a person stand in front of camera, stand still for 5 seconds and perform some movements for another 5 seconds.

Test result: the output file saves and is seemingly correct at this point. We would have to test with a tape measure as reference and go back to check the saved video to be completely certain.

Apple Watch - No of Spins

We have a method using the gyroscope and accelerometer inside the apple watch to identify the number of turns the person performed.

Test method: having a person performs turning with different angle and speed and compared result with the amount of turing actually happened.

Test result: In most condition the turns are recognised properly, however when you are turning too fast, the method result appears to be higher than the actual result.
